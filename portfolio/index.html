Преимущества

  • Валидность, кроссбраузерность

    Корректное отображение макета во всех браузерах

  • Семантический
    HTML-код

    Использовании HTML-тегов в соответствии с их семантикой (предназначением), а также логичная и последовательная иерархия страниц

  • Адаптивность

    Правильное отображение сайта на различных устройствах, подключённых к интернету и динамическая подстройка под заданные размеры окна браузера

  • Mobile First

    Mobile First это метод разработки оптимизированного вебсайта для различных мобильных устройств с учетом скорости подключения к сети. Такой метод позволяет отображать основное содержание конечному пользователю

  • Flex

    Flexbox — это новый способ располагать блоки на странице.
    С помощью Flexbox можно легко выравнивать элементы по горизонтали и по вертикали, менять направление и порядок отображение элементов, растягивать блоки на всю высоту родителя или прибивать их к нижнему краю.

  • Адаптированная графика

    Ретинизация изображений, svg, спрайты и прогрессивный jpg для удобного просмотра

  • Методология БЭМ

    Методология БЭМ создана в Яндексе для разработки сайтов, которые надо делать быстро, а поддерживать долгие годы. Она позволяет создавать расширяемые и повторно используемые компоненты интерфейса.

  • Анимация CSS и JS

    Все что нужно для визуальной динамики на Вашем сайте. Интерактивные всплывающие меню и формы, слайдеры, оживленные декоративные элементы и пр.
